## How to Deploy

This website is designed to be deployed on GitHub Pages. Simply push the files to your GitHub repository and enable GitHub Pages in your repository settings.

### Steps:
1. Push all files to your `vision.github.io` repository
2. Go to your repository Settings
3. Scroll down to "GitHub Pages"
4. Select the main branch as your publishing source
5. Your site will be live at `https://yourusername.github.io`

## Customization

### Update Company Information
Edit `index.html` to update:
- Contact information in the Contact section
- Social media links
- Team members
- Company description

### Modify Colors
Edit `styles.css` to change the color scheme. Key colors are defined in the `:root` CSS variables:
- `--primary-color`: Main background color
- `--secondary-color`: Secondary background
- `--highlight-color`: Accent/highlight color
- `--success-color`: Success/info color

### Add More Team Members
Simply duplicate the team member blocks in `index.html` and customize the content.
